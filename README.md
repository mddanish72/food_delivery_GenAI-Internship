## Food Delivery Analytics Project

**Overview:**  
Built an end-to-end food delivery analytics dataset by integrating CSV, JSON, and SQL data sources using Python (**Pandas & SQLite**).

**Key Highlights:**
- **Data Cleaning & Preparation:** Cleaned data, performed joins, and engineered features for analysis.
- **Integrated Multiple Sources:** Combined CSV, JSON, and SQL data for a unified dataset.
- **Analysis & Insights:** Enabled exploration of user behavior, restaurant performance, and revenue trends.
